Definition List

Information
Meaningful words, numbers, pictures and sounds.
File
A logical collection of information stored under a single name.
Directory
A logical collection of files stored under a single name.
Primary Storage
A volatile and temporary form of storage which can be directly accessed by the CPU. An example of this is Random Access Memory. This type of storage is fast and relatively expensive.
Secondary Storage
A stable and relatively permanent form of storage which is cheap but slow. Examples of this include, CD, DVD, hard drive, etc.
JPEG
The abbreviation of JPEG is joint photographic experts group. It is a widely used file format for images.
GIF
The abbreviation of GIF stands for the graphics interchange format. Files of this type have moving images (animation) without sound.
PNG
The abbreviation of PNG is portable network graphics. PNG files are image type files that can have different levels of transparency.
PDF
The abbreviation of PDF stands for portable document format. This file format allows all elements of a printed document to be captured into an image or multiple images which can be viewed, navigated or printed.
